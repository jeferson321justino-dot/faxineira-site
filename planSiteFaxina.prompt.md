# Plano: Site de Faxina para Sua Esposa

## Visão Geral
Criar um site one-page (página única) gratuito em HTML/CSS com botão flutuante de WhatsApp para conversão de clientes.

## Estrutura do Site

### Seções da Página
1. **Header** - Logo/nome + navegação suave
2. **Hero** - Título chamativo + subtítulo + CTA WhatsApp
3. **Serviços** - Cards com serviços oferecidos
4. **Sobre** - Breve apresentação da profissional
5. **Depoimentos** - Clientes satisfeitos (opcional)
6. **Contato** - Botão WhatsApp + formulário simples
7. **Footer** - Direitos autorais + links sociais

## Passos de Implementação

### Fase 1: Estrutura HTML
- Criar arquivo `index.html`
- Definir semântica (header, section, footer)
- Adicionar meta tags para SEO local

### Fase 2: Estilização CSS
- Criar arquivo `style.css`
- Design responsivo (funciona no celular)
- Cores suaves e profissionais (azul/verde limpeza)
- Tipografia legível

### Fase 3: Funcionalidades
- Botão flutuante WhatsApp (fixo na tela)
- Animações sutis nos elementos
- Links suaves entre seções

### Fase 4: Publicação
- Opção: GitHub Pages (gratuito)
- Ou: Netlify (gratuito)

## Arquivos a Criar
- `index.html` - Estrutura principal
- `style.css` - Estilos
- `script.js` - Funcionalidades (opcional)

## Verificação
1. Site carrega corretamente
2. Botão WhatsApp funciona
3. Design responsivo (mobile/desktop)
4. Links de navegação funcionam
5. Imagens carregam

## Decisões Tomadas
- ✅ One-page para simplicidade
- ✅ Botão WhatsApp como principal conversão
- ✅ HTML/CSS básico sem custos
- ✅ Foco em faxina residencial
